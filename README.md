# 🔊 Text to Voice Converter

A simple and responsive web application that converts text into spoken audio using the Web Speech API. Built with HTML, CSS, and JavaScript.

---

## 🚀 Features

- Convert any text into speech
- Choose from available system voices
- Fully responsive UI
- Clean and modern design using CSS and Bootstrap Icons
- Works directly in modern browsers (no back-end required)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Web Speech API)
- Bootstrap Icons

---

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/text-voice-convertor.git
   cd text-voice-convertor

2. Install Dependencies:

    npm install

  This installs bootstrap-icons used in the UI.

✅ Usage

      1.Open index.html in any modern browser.
      2.Type or paste any text.
      3.Select a voice from the dropdown.
      4.Click "Listen" to hear it spoken aloud.

🌍 Browser Support

      Chrome ✅
      Edge ✅
      Firefox ✅ (limited voices)
      Safari ✅

      ⚠️ In Chrome, voice list may load asynchronously. The app handles this via onvoiceschanged.
